# Product Overview

This is a **Todo Management SaaS** application built with Next.js. The application provides task management capabilities with support for:

- **Multi-tenant architecture** - Each user has their own tenant space
- **GTD (Getting Things Done) methodology** - Optional GTD workflow support
- **Comprehensive task organization** - Tasks can be organized by Projects, Contexts, Areas, and Tags
- **Authentication system** - Email/password authentication with NextAuth.js
- **User preferences** - Customizable user settings and preferences
- **Inbox processing** - GTD-style inbox for capturing and processing items

## Core Features

- Task creation, editing, and completion
- Project management with outcomes and next actions
- Context-based task organization (@calls, @computer, etc.)
- Area-based life organization (Work, Personal, etc.)
- Tag-based categorization
- Due date management with overdue/today/upcoming views
- Search and filtering capabilities
- User preference management
- Password reset functionality

## Target Users

Individuals and teams looking for a structured approach to task and project management, with optional GTD methodology support.
